# Repositorio de Práctica DevOps

Este repositorio contiene ejercicios prácticos desarrollados como parte del curso de **DevOps**.

El objetivo es reforzar conceptos clave como integración continua, control de versiones, automatización y buenas prácticas de desarrollo.

## Contenido

- Ejercicios prácticos de DevOps
- Uso de Git y GitHub
- Implementación de pipelines CI con GitHub Actions
- Validaciones automáticas en Pull Requests

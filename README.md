# Repositorio de Práctica DevOps

Este repositorio contiene ejercicios prácticos desarrollados como parte del curso de **DevOps**.

El objetivo principal es reforzar conceptos clave como integración continua, control de versiones, automatización de procesos y buenas prácticas de desarrollo de software.

## Contenido

- Ejercicios prácticos orientados a DevOps
- Uso de Git y GitHub para control de versiones
- Implementación de pipelines de Integración Continua (CI) con GitHub Actions
- Validaciones automáticas en Pull Requests mediante herramientas de calidad

## Objetivo del Repositorio

Servir como un espacio de práctica para aplicar conceptos DevOps en un entorno controlado, fomentando el uso de estándares, automatización y revisión continua del código.
